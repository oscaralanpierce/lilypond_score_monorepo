# Lilypond Score Monorepo

As a musician, keeping track of my scores has begun to be a pain point. I like to engrave scores with [LilyPond](https://lilypond.org) and that makes them great candidates to be stored in a monorepo on GitHub
instead of 40,000 3-ring binders taking up space in my house.

This monorepo will include LilyPond files to build scores of my arrangements of a lot of the music I have performed.

## Note on Intellectual Property

Although I am making this repository public, the original copyright holders (usually not me) retain rights to the music contained within. As the arranger, I also retain rights to the arrangements to the extent allowed by law. I am mainly creating this repo for my own convenience but I like sharing. If that ends up causing me problems with music publishers I'll make this repo private.
